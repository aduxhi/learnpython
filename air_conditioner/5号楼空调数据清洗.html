清洗算法：scipy-signal工具包中的prominence(突出点)算法¶

原理：计算信号中每个峰值的突出程度。 峰值的突出度衡量峰值从信号的周围基线突出多少，并定义为峰值与其最低等值线之间的垂直距离

from scipy import signal
import numpy as np
import copy
In [15]:
ia = oridata['ia']
iaa = copy.deepcopy(ia)

for i in range(5):
    peaks, _ = signal.find_peaks(iaa, height = ia.mean(), width=[0,4], prominence=8)
    iaa[peaks] = iaa[peaks-1]
